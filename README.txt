A free and simple WordPress plugin for MailBluster which provides different methods to create and include subscription forms into WordPress pages or posts by utilizing AmazonSES service.


== Description ==

[MailBluster](https://mailbluster.com) is the most advanced, reliable and cost-effective email marketing software in the world. It's so capable; you can send promotional email to millions while keeping every single email entirely personalized.

[MailBluster](https://mailbluster.com) WordPress plugin is a simple and free software for creating subscription forms for the [MailBluster](https://mailbluster.com) email campaign service. It provides different methods to create and include subscription forms to WordPress pages or posts via API utilizing AmazonSES service.

For more info about linking your WordPress site with MailBluster you can read [this](https://mailbluster.com/blog/how-to-link-wordpress-site-with-mailbluster) blog

* **List segmentation.** Target and re-target subscribers based on who opened (or didn't open) a specific email, clicked a certain link in an email or what products they bought.

== Installation ==

1. In your WordPress admin panel, go to Plugins > New Plugin, search for ‘MailBluster For WordPress’ and click 'Install now'. Alternatively, download the plugin and upload the contents of mailbluster4wp.zip to your plugins directory, which is usually located at '/wp-content/plugins/'.
2. Activate the plugin and enter your API key from [MailBluster App](https://app.mailbluster.com).
3. Create a form using the built-in form builder.
4. Enter a short code onto a page or post or custom post type, or use the MailBluster widget to select a form to display.
